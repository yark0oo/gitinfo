Клонирование проекта в папку nameProject:
git clone https://... nameProject

добавить файлы в коммит:
git add .

записать коммит:
git commit -m "Комментарий коммита"

отправить коммит:
git push

принять изменения:
git pull

Авторство коммитов:
git config --global user.email "Ваш емаил"
git config --global user.name "Ваше имя"
